Odoo Marketplace Seller Slider
Seller can set image slider on their profile page and shop page.

Note: This module is an addon for Odoo Multi Vendor Marketplace module. The price appearing here is inclusive of all the modules (addon + dependent modules) cost. In case, you are already owning any of the dependent modules then its respective cost will be deducted from the total cost mentioned above.

Features:

  • Seller can set image slider on their profile page and shop page.
  • Seller can upload multiple images for slider.
  • Seller can set or change properties of each image like URL, sequence, etc.
  • Seller can delete the uploaded images of his slider.
  • Seller can hide or show images in slider.

  • Work-flow
