Website Guest Checkout

Easy way to register Your Guest Before Checkout in Odoo Website!

Odoo Website Guest Checkout allows you to apply restrictions on guest checkout without login / sign up, which is by default supported on odoo website. Guest Users can purchase a product simply by signing in the checkout page.
